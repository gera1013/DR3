# DR3
Implementación de Ray Tracing para simular un cuarto.

## Ejecución
1. Descargar archivos
2. Ejecutar el archivo con el nombre drthree.py
3. Al finalizar la ejecución se creará un archivo bmp con el nombre output.bmp
